# 🎯 GitHub 推送快速指南（复制粘贴版）

## ⚡ 三行命令搞定

```powershell
git remote add origin https://github.com/YOUR_USERNAME/solar-system-360.git
git branch -M main
git push -u origin main
```

---

## 📋 完整步骤

### 步骤 1️⃣ - 创建 GitHub 仓库
1. 访问 https://github.com/new
2. **Repository name**: `solar-system-360`
3. **Description**: `A realistic 360° Solar System visualization with Three.js`
4. **Visibility**: 选择 **Public**
5. ❌ 不勾选 "Initialize this repository with"
6. 点击 **Create repository**

### 步骤 2️⃣ - 复制仓库 URL
- GitHub 会显示你的仓库 URL
- 复制 HTTPS 版本：`https://github.com/YOUR_USERNAME/solar-system-360.git`

### 步骤 3️⃣ - 在 PowerShell 中执行
```powershell
cd C:\Users\User\Desktop\cop
git remote add origin https://github.com/YOUR_USERNAME/solar-system-360.git
git branch -M main
git push -u origin main
```

### 步骤 4️⃣ - 输入凭证
- GitHub 用户名或邮箱
- GitHub 密码或 Personal Access Token

### 步骤 5️⃣ - 验证成功
访问 `https://github.com/YOUR_USERNAME/solar-system-360` 查看你的仓库
