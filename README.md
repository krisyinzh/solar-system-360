# 🌌 宇宙·太阳系 360° 可视化

一个高保真、360°可旋转的太阳系可视化项目，采用 Three.js WebGL 渲染，具有逼真的行星纹理、动态光照和交互功能。

## ✨ 主要特性

- **🌟 40,000 颗逼真星星** - 基于哈兹普龙-罗素图真实分布
- **🪐 8 颗行星 + 月球** - 水星、金星、地球、火星、木星、土星、天王星、海王星
- **📏 真实大小比例** - 以太阳为基准的科学相对大小
- **💫 高保真渲染** - PBR 材质、1024×1024px 程序纹理、法线贴图、动态阴影
- **🎯 交互信息** - 点击行星/太阳查看详细科学信息
- **📍 实时标签** - 所有天体上方显示标签，跟随屏幕位置
- **🎮 完整控制** - 鼠标拖拽旋转、滚轮缩放、空格切换轨道
- **⚡ 动态模拟** - 真实公转周期、初始位置分散、轨道倾斜

## 🚀 快速开始

### 方式一：直接打开
1. 双击 `index.html` 在默认浏览器中打开
2. 或用任何现代浏览器（Chrome、Firefox、Safari、Edge）打开

### 方式二：本地服务器（推荐）
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (http-server)
npx http-server
```
然后访问 `http://localhost:8000`

## 🔧 技术栈

- **Three.js r128** - WebGL 3D 图形库
- **OrbitControls** - 摄像机交互控制
- **Canvas 2D** - 程序化纹理生成
- **Raycaster** - 鼠标拾取检测
- **PBR 材质** - 物理基渲染（金属度、粗糙度、法线贴图）

## 📊 渲染优化

- 256×256 球体细分（高质量表面）
- 1024×1024 程序纹理和法线贴图
- 4096×4096 PCF 阴影映射
- ACES Filmic 色调映射
- 40,000 点粒子星场
